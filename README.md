# Crisis Atlas

**Mapping the 21st Century's Interconnected Crises**

An interactive, data-driven atlas tracking nine interconnected crisis dimensions across the globe: climate change, pollution, poverty & inequality, cybercrime, food & water security, conflict & displacement, pandemic preparedness, biodiversity loss, and democratic backsliding.

## Features

- **🌐 Interactive World Map** — Choropleth visualization of crisis intensity by country
- **📊 9 Crisis Dimensions** — Deep-dive panels with country rankings, KPI cards, and bar charts
- **🔗 Cross-Crisis Correlations** — Pearson correlation matrix revealing how crises amplify each other
- **📡 Live API + Static Fallback** — World Bank API integration with curated fallback data
- **🎨 Economist Aesthetic** — Clean, data-driven design with Crimson Text typography
- **📱 Responsive** — Adapts from desktop to tablet

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS v4** (styling)
- **Recharts** (data visualization)
- **React-Leaflet** (interactive maps)
- **World Bank API** (live data)

## Data Sources

World Bank, WHO, FAO, ITU, IUCN, EIU, UNHCR, ND-GAIN, Transparency International, Reporters Without Borders, and Global Health Security Index.

## Development

```bash
npm install
npm run dev      # Start dev server
npm run build    # Production build to dist/
```
